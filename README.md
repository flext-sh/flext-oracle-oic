# FLEXT Oracle OIC

Biblioteca de integracao com Oracle Integration Cloud para operacoes de conectividade e interoperabilidade.

Descricao oficial atual: "FLEXT Oracle OIC Extension - Advanced Oracle Integration Cloud Extensions".

## O que este projeto entrega

- Padroniza chamadas para recursos de integracao cloud.
- Apoia automacoes orientadas ao dominio Oracle.
- Encapsula consumo de OIC para reuso no portfolio.

## Contexto operacional

- Entrada: requisicoes e credenciais Oracle OIC.
- Saida: operacoes de integracao executadas no ambiente cloud.
- Dependencias: acesso ao tenant OIC e configuracao de seguranca.

## Estado atual e risco de adocao

- Qualidade: **Alpha**
- Uso recomendado: **Nao produtivo**
- Nivel de estabilidade: em maturacao funcional e tecnica, sujeito a mudancas de contrato sem garantia de retrocompatibilidade.

## Diretriz para uso nesta fase

Aplicar este projeto somente em desenvolvimento, prova de conceito e homologacao controlada, com expectativa de ajustes frequentes ate maturidade de release.
